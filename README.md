Введение
Настоящее руководство оператора содержит исчерпывающее описание порядка работы с прикладным приложением «JUST IN SPACE». В документе детально рассмотрены основные функциональные возможности программы, интерфейс пользователя и пошаговый алгоритм действий для проведения успешной симуляции. Дополнительно приведены рекомендации, позволяющие эффективно использовать все возможности приложения.

Используемые обозначения и сокращения
Для однозначного понимания материала в данном руководстве используются следующие термины:

Симуляция полета – ключевой процесс работы приложения, включающий в себя:

Расчет динамических параметров космической ракеты (время полета, высота полета, скорость полета, количество оставшегося топлива);

Графическую визуализацию космической ракеты в реальном времени с отображением рассчитываемых параметров.

О продукте
Приложение «JUST IN SPACE» представляет собой интерактивный симулятор, предназначенный для моделирования полета космической ракеты в условиях, приближенных к земным и околоземным. Пользователь получает возможность задавать ключевые параметры ракеты-носителя и наблюдать за этим в наглядной графической форме.

Алгоритм работы:

Для перехода от окна входа (Рисунок 1) необходимо нажать кнопку «Старт», расположенную в нижней части экрана.

После открытия главного окна необходимо задать для каждой из трёх ступеней ракеты следующие параметры:

Сухая масса

Масса топлива

Расход топлива

После ввода параметров пользователь может запустить симуляцию, нажав кнопку «Запустить ракету». Процесс будет отображаться в меню симуляции (Рисунок 3).

Осуществление ввода данных
Ввод всех необходимых параметров для симуляции осуществляется пользователем через текстовые поля в меню, показанном на Рисунке 2.
Система автоматически проверяет корректность введённых данных (допускаются только положительные числовые значения). После ввода всех данных симуляция запускается при нажатии на кнопку «Запустить ракету».

Вывод информации
В процессе выполнения симуляции приложение предоставляет пользователю результаты в двух формах:

Текстовый вывод – отображаются значения динамически изменяемых параметров ракеты.

Графический вывод – визуализация изменения положения ракеты в пространстве, имитирующая взлёт и набор высоты.
