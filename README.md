# Введение

Настоящее руководство оператора содержит исчерпывающее описание порядка
работы с прикладным приложением «JUST IN SPACE». В документе детально
рассмотрены основные функциональные возможности программы, интерфейс
пользователя и пошаговый алгоритм действий для проведения успешной
симуляции. Дополнительно приведены рекомендации, позволяющие эффективно
использовать все возможности приложения.

# Используемые обозначения и сокращения

Для однозначного понимания материала в данном руководстве используются
следующие термины:

a.  Симуляция полета -- ключевой процесс работы приложения, включающий в
    себя:

    a.  Расчет динамических параметров космической ракеты (время полета,
        высота полета, скорость полета, количество оставшегося топлива);

    b.  Графическую визуализацию космической ракеты в реальном времени с
        отображением рассчитываемых параметров;

# О продукте

Приложение «JUST IN SPACE» представляет собой интерактивный симулятор,
предназначенный для моделирования полета космической ракеты в условиях,
приближенных к земным и околоземным. Пользователь получает возможность
задавать ключевые параметры ракеты-носителя и наблюдать за этим в
наглядной графической форме.

# Осуществление ввода данных

Ввод всех необходимых параметров для симуляции осуществляется
пользователем через текстовые поля в меню, показанном на рисунке 2.

Система автоматически проверяет корректность введенных данных
(допускаются только положительные числовые значения). После ввода всех
необходимых данных симуляция запускается при нажатии на кнопку
«Запустить ракету».

# Вывод информации

В процессе выполнения симуляции приложение предоставляет пользователю
комплекс результатов в двух формах:

-   Текстовый (выводятся значения динамически изменяемых параметров
    ракеты);

-   Графический (изменение ракеты в пространстве, имитируя взлет, набор
    высоты);
